最近鼠标左键又不行了，单击时不时变双击，以前有过两次，每次都是拆开微动，将触电氧化部分擦一擦就又能用几个月。这次实在是忍不了了，又不想换鼠标，毕竟跟了我几年，用的也非常之顺手（其实就是没钱换新的），于是淘宝了一套工具，自己来换个微动，应该就又能用几年了。

安装 Redshift，解决首次启动自动退出问题

发表于 2015-09-10 | 分类于 技术 |

Redshift 是我个人非常喜欢的一款软件，作用是根据环境（也就是根据你当前所处经纬度换算得来的时间）调整屏幕的色温。具体表现就像是在屏幕上覆盖上了一层朦胧的红光，随着夜色来临，红光的强度也会越来越大，当然，这都是可调的。
说的这么高大上，其实通俗点来讲，作用就是“去蓝光”！

源码可见：Redshift Github

已集成在Ubuntu官方源里，所以可以直接安装：

sudo apt-get install redshift-gtk

启动后，勾选’autostart’，会在’start applications’里面出现’redshift’的启动项，现在便会开机自启了。

现在还不算完，你会发现在开机后 redshift 自动启动几秒后又会自动退出，经 Google 搜索得出是因为 geoclue 冲突，导致 redshift 读不出位置信息，所以FC了，原文参考How do I get redshift to start with my computer?

VPS 安装锐速，提升传输速度到极致

发表于 2015-09-02 | 分类于 技术 |

**2016-10-02更新：锐速已经停止注册了，现在有高人放出来锐速破解版，安装使用更加简单了，只不过内核需要匹配。
锐速破解版 github 直达

实测环境：VPS为 Vultr，Kvm 架构，Ubuntu 14.04 x64系统。

首先要VPS所安装的系统版本和内核版本符合锐速支持，具体可在锐速当前支持的系统与内核版本查看。

现在VPS提供的系统一般内核都是比较新的版本，不受锐速的支持，比如我在 Vultr 安装的 Ubuntu 14.04 x64 的内核版本就是 3.13.0-52，而从上面的网站查阅可知，锐速在该版本的的 Ubuntu 中支持的内核最高版本为 3.13.0-48，所以就需要更换内核才能安装，不然会提示安装失败！
