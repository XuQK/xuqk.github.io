Android Studio 不能同步代码的坑

发表于 2016-11-16 | 分类于 技术 |

今天写码正写的嗨的时候，就给我来状况了，在 AS 里更新了几行代码，Run 的时候在模拟器上没有预期的效果。

反复检查了代码，打了各种断点，硬是没发现改的那几行代码有发挥任何作用的迹象，多方求助无果。因为正学习到《Android 权威指南》的“数据库”这章，想起里面有句话“记住，在本章学习过程中，如遇数据库相关问题，可尝试直接删除应用，重头来过”，无奈之下只能照做，结果特么的删除重新 Run 之后，改的代码就生效了！

就在我把解决方法往群里一贴，只见群里大神Karl来了句“AS 有时候不能同步代码很正常”，此时我心中真是一万头草泥马奔腾而过，尼玛的现在来放马后炮，害我折腾那么久！

不管怎么说，重装真是治 BUG 利器~
