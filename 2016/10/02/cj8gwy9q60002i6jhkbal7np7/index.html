Android Studio 2.2 中对 lambda 语法的支持问题

发表于 2016-10-02 | 分类于 技术 | | 你是第 个瞅这篇文章的哟

今天学习 Android 开发的时候，发现 Android Studio 2.2 对 jdk8 的 lambda 支持一个很奇怪的地方，AS 只允许 IDE 本身对代码执行 lambda 形式的代码转换，而对开发者手动输入的 lambda 语句会给出 ‘lambda are not supported at this language level’。

Google 了一些方法，里面都说要将 AS 的 jdk 版本替换为 jdk8，但是我本身就是使用的 jdk8，所以并没有得到什么帮助。然后我索性用手动输入的 lambda 语句直接在测试机上运行，但是执行过程中报给了我一个这样的错误：

这特么就很尴尬了，我明明用的是 1.8,为什么运行的时候加载的是 1.7呢？Google 一下，并没有找到什么有用的东西，这个问题，还是留待以后再说吧。
