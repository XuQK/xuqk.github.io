Firefox 也能用上最新版 flash

发表于 2015-07-12 | 分类于 技术 | | 你是第 个瞅这篇文章的哟

2015年7月31日更新：试用一个月，2周前开始Firefox开始频繁崩溃，每次关闭Firefox必定报错，使用途中也时不时报错，换回 Ubuntu Software Center 中的 flash 后恢复正常。

大意是让Ubuntu下的浏览器使用集成在Chrome里面的Flash这样。
而这个版本是转为Firefox准备的，不保证其余浏览器里面能顺利运行。

然后作者balabala讲了一大堆，大意是这只是个测试版本，所以使用一定要慎重什么的。不过据我自己测试没发现有什么致命的BUG。

安装：

sudo add-apt-repository ppa:nilarimogard/webupd8
sudo apt-get update
sudo apt-get install freshplayerplugin

然后由于这个flash是Chrome里面的版本，所以依赖“libpepflashplayer.so”这个库文件运行。而安装这个文件最简单的方式就是直接安装Chrome浏览器，但我是用火狐的，又有强迫症，所以还是选择单独安装该文件，Ubuntu14.04的官方源里就有，所以直接安装：

下载包：

sudo apt-get install pepperflashplugin-nonfree

安装：

sudo update-pepperflashplugin-nonfree --install

要是你想安装beta版本的：

sudo update-pepperflashplugin-nonfree --install --beta --unverified

要是想安装Unstable版的：

sudo update-pepperflashplugin-nonfree --install --unstable --unverified

卸载命令：

sudo update-pepperflashplugin-nonfree --uninstall

这里是pepperflash配置文件模板 。重命名为“freshwrapper.conf”放到“~/.config”下面。

若是发现视频无法播放，说找不到“libpepflashplayer.so”的路径，那么去系统路径下面找到这个文件，我的是在“/usr/lib/pepperflashplugin-nonfree/libpepflashplayer.so”，然后去配置文件里找到“pepperflash_path”这一行，改一下就OK了。
