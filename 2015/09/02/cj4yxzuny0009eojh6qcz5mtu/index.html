VPS 安装锐速，提升传输速度到极致

发表于 2015-09-02 | 分类于 技术 | | 你是第 个瞅这篇文章的哟

**2016-10-02更新：锐速已经停止注册了，现在有高人放出来锐速破解版，安装使用更加简单了，只不过内核需要匹配。
锐速破解版 github 直达

实测环境：VPS为 Vultr，Kvm 架构，Ubuntu 14.04 x64系统。

首先要VPS所安装的系统版本和内核版本符合锐速支持，具体可在锐速当前支持的系统与内核版本查看。

现在VPS提供的系统一般内核都是比较新的版本，不受锐速的支持，比如我在 Vultr 安装的 Ubuntu 14.04 x64 的内核版本就是 3.13.0-52，而从上面的网站查阅可知，锐速在该版本的的 Ubuntu 中支持的内核最高版本为 3.13.0-48，所以就需要更换内核才能安装，不然会提示安装失败！

下面开始具体操作

1. 注册锐速帐号

在锐速会员中心注册个帐号，就可以使用它的免费套餐了，适用于 20Mbps 带宽的的服务器加速。虽然现在的 VPS 基本都是 100Mbps 带宽起，但是我们普通用户能有实际上的 20Mbps 能用就相当不错了，所以免费套餐基本够用。

2. 更换系统内核

注意：此方法仅在 Ubuntu 测试过，其余系统更换内核方法请自行 Google！

Ubuntu 更换内核是超级简单的！

首先安装需要的内核（在前面提到的网站查阅可知）

sudo apt-get install linux-image-3.13.0-46-generic

然后卸载不需要的内核，切换内核版本

sudo dpkg --get-selections | grep linux-image //列出系统中已有的内核
sudo apt-get remove linux-image-3.13.0-53-generic linux-image-3.13.0-62-generic //删除不需要的内核
sudo update-grub //重建 Grub 引导文件
sudo reboot now //重启
sudo apt-mark hold linux-image //重启之后还需要固定下内核版本，避免意外升级。

3. 安装锐速

wget http://my.serverspeeder.com/d/ls/serverSpeederInstaller.tar.gz //下载锐速安装脚本
tar xzvf serverSpeederInstaller.tar.gz //解压脚本
sudo bash serverSpeederInstaller.sh //运行安装脚本

根据屏幕提示输入之前注册的锐速帐号密码，然后后还会提示你输入诸如“欲加速的接口”、“上下行带宽”这些参数，一般来说默认的就好。最后会让你选择是否随系统自动运行，毫无疑问选是。

然后就享受极致带宽吧！

4. 博主使用感受

之前使用在 Vultr 上自建的 Shadowsocks 的时候，速度总是不稳定，从 Google Play 下载软件基本是 20K/s 的速度上下浮动，Youtube 更是完全不要想了。
安装锐速后，效果立竿见影，Play 下载软件没低于过 1M/s，大多数时间是 2M/s 左右，Youtube 720P 基本上没卡过（对1080P没需求，所以没测试过，应当差不多），缓冲速度也是 1M/s+。

注意：安装后几分钟到几十分钟内可能 VPS 的 mtr 结果会很不好，具体表现为延迟变得非常大且很不稳定，我的 Vultr 安装锐速前延迟大概在 160ms 左右，安装后几分钟内测试，从 300ms 到 2000ms不等，过个是几分钟就会稳定下来了，现在稳定在 190ms 左右，丢包率在 0.1% 左右。所以安装完后先试用个一天看效果，千万别刚安装看效果不行就马上卸载咯！
